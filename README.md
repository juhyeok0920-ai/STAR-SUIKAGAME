# ✨ STAR-SUIKAGAME (스텔라 에볼루션)

> **"철(Fe)을 생성하고 초신성 폭발을 일으켜 우주를 확장하세요!"**
> Matter.js 물리 엔진과 Firebase 실시간 데이터베이스를 활용하여 구현한 웹 기반 우주 테마 수박게임입니다.

<br>

## 🌌 프로젝트 소개
기존의 과일 합성 게임(수박게임)을 **우주 원소의 진화(Stellar Evolution)** 테마로 재해석한 물리 퍼즐 게임입니다. 유저는 수소(H)부터 시작해 원소들을 결합하며 최종 단계인 철(Fe)을 형성해야 합니다. 철과 철이 만나면 거대한 **초신성 대폭발(Supernova)**이 일어나며 게임에서 승리하게 됩니다.

* **배포 주소:** [본인의 Vercel 배포 링크를 적어주세요]

<br>

## 🚀 주요 기능

### 1. 물리 엔진 기반의 원소 합성 시스템
* **Matter.js** 라이브러리를 활용하여 리얼한 중력, 마찰력, 반발계수를 가진 원소 드롭 및 충돌을 구현했습니다.
* 동일한 원소가 충돌하면 다음 단계의 고질량 원소로 융합(Merge)되며 점수를 획득합니다.
* 캔버스 트랜스폼 연산을 통해 물리 회전 각도에 맞춰 원소 기호(`H`, `He`, `C` 등)가 함께 회전하도록 디테일을 살렸습니다.

### 2. 하드코어 우주 붕괴 시스템 (데드라인 제어)
* 컨테이너 상단에 임계점 가이드라인(2.00초 시간 제한)이 존재합니다.
* 원소가 선을 넘으면 UI가 붉게 활성화되며 실시간 카운트다운이 시작됩니다. 2초 동안 선 아래로 원소를 정돈하지 못하면 중력 균형이 깨지며 **블랙홀 엔딩(Mission Failed)**으로 게임이 종료됩니다.

### 3. Cloud Firestore 연동 실시간 글로벌 랭킹 (Top 10)
* 게임 오버 또는 초신성 성공 시 유저의 최종 점수와 닉네임이 **Firebase Firestore**에 즉시 보안 저장됩니다.
* 메인 화면의 `GLOBAL TOP 10` 패널을 통해 전 세계 플레이어들의 최고 기록을 실시간(Real-time Query)으로 확인할 수 있습니다.

### 4. 비로그인 기반 커스텀 닉네임 시스템
* 별도의 회원가입 없이 브라우저 로컬 스토리지(`localStorage`)를 이용해 고유 유저 ID를 유지합니다.
* 유저가 직접 원하​​는 닉네임(영문/숫자 10자 이내)을 설정할 수 있으며, 설정된 닉네임으로 랭킹보드에 기록됩니다.

### 5. 완벽한 3단 와이드 레이아웃 (반응형 최적화)
* 화면 잘림 현상을 해결하기 위해 **[왼쪽: 진화 가이드 | 가운데: 게임판 | 오른쪽: 스코어 및 랭킹]** 구조의 3단 독립 대시보드 레이아웃을 구현했습니다.
* 세로 스크롤 없이 한 화면에 모든 게임 요소와 후원 버튼이 한눈에 쏙 들어오도록 UI 종속성을 완벽히 조율했습니다.

### 6. 소셜 공유 및 후원 연동
* 게임 결과 화면에서 **𝕏(구 트위터) 실시간 공유** 및 게임 링크 복사 기능을 제공합니다.
* 개발자 지원을 위한 글로벌 후원 플랫폼 **Buy Me a Coffee** 연동 버튼이 포함되어 있습니다.
